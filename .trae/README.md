# 🤖 Trae AI Development Workspace

> 大模型应用开发工作空间

## 📋 项目简介

本目录用于存放大模型应用开发相关的代码、配置和规则文件。这是一个专注于 AI 应用开发的工作空间，旨在提供结构化的开发环境和最佳实践。

## 🎯 主要功能

- **大模型应用开发**: 集中管理和开发基于大语言模型的应用程序
- **代码规范管理**: 通过自定义规则确保代码质量和一致性
- **开发工作流优化**: 提供标准化的开发流程和工具配置

## 📁 目录结构

```
.trae/
├── README.md           # 项目说明文档
└── rules/              # 开发规则和配置
    └── git-commit-message.md  # Git 提交消息规范
```

## 🚀 快速开始

1. 克隆仓库到本地
2. 查看 `rules/` 目录下的配置文件
3. 根据项目需求调整规则配置

## 🛠️ 技术栈

- 大语言模型 (LLM)
- AI 应用开发框架
- 自动化工具链

## 📝 开发规范

本项目遵循以下开发规范：

- Git 提交消息规范 (详见 `rules/git-commit-message.md`)
- 代码风格统一
- 文档完整性要求
